Markup Language
---------------

`reStructuredText <https://en.wikipedia.org/wiki/ReStructuredText>`_ (.rst) lightweight markup language is used for the purpose of writing this development documentation. The source files reside in `LibreClinica-docs <https://github.com/reliatec-gmbh/LibreClinica-docs>`_  repository. This repository contains a webhook to trigger the re-build of static HTML resources that are publicly hosted via `Read the Docs <https://readthedocs.org/>`_ (RTD) utilising RTD Sphinx Theme.
